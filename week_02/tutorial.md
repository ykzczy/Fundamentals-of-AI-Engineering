# Week 2 Tutorials: IDE + AI-Assisted Code Practice

## Overview

Week 2 transitions from browser-based AI tools to using AI directly within an IDE environment. You'll learn to use AI assistants for code reading, modification, and debugging — without needing to write code from scratch.

You will practice:

- Setting up VS Code or Cursor with AI integrations
- Using AI to read and understand unfamiliar code
- Using AI to modify existing code
- Using AI to debug common errors

## Recommended order

1. Read 01 and set up your IDE with AI integration.
2. Read 02 to understand the AI-assisted workflow.
3. Read 03 and practice explaining code with AI.
4. Read 04 and practice modifying code with AI.
5. Read 05 and practice debugging with AI.

Why this order works:

1. **Environment first**
    - Your IDE must be properly configured before you can practice.
    - What to verify: you can open Cursor, see AI chat panel, and send a prompt.

2. **Workflow second**
    - Understand the pattern before practicing specific skills.
    - What to verify: you can explain the "ask → review → apply" pattern.

3. **Reading third**
    - Learn to understand code before modifying it.
    - What to verify: you can have AI explain at least 5 functions from templates.

4. **Modification fourth**
    - Now you can make changes with understanding.
    - What to verify: you can successfully modify at least 3 functions.

5. **Debugging fifth**
    - Errors are inevitable — learn to handle them.
    - What to verify: you can use AI to understand and fix at least 2 errors.
